在 CMD 里跑起来（Windows）
1) 打开 CMD
Win + R → 输入 cmd → 回车
2) 进入项目目录
cd /d F:\outlookEmailPlus
3) 创建并激活 venv
python -m venv .venv
.\.venv\Scripts\activate.bat
看到提示符前面变成 (.venv) 就表示激活成功。

4) 安装依赖
python -m pip install -r requirements.txt
5) 启动服务（带必要环境变量）
set SECRET_KEY=your-secret-key-here
set LOGIN_PASSWORD=12345678
python web_outlook_app.py
如果你想自动生成 SECRET_KEY（可选），可以在 CMD 里这样做：

for /f %i in ('python -c "import secrets;print(secrets.token_hex(32))"') do set SECRET_KEY=%i
6) 打开页面
浏览器访问：

http://127.0.0.1:5000/
